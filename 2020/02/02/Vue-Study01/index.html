Vue Study01

发表于 2020-02-02 更新于 2020-02-04

Vue 第一天


var app = new Vue({
            el: '#app', 
            data:{
                content:'Hello World!',
            }
        })

理解

el: 代表这个vue的实例接管的区域 #app 就是 id 为 app的区域
data: 定义一些数据

Vue 简单修改数据

setTimeout(function(){
            app.$data.content = "Bye World"
        },2000)

理解

模块名.$data.数据变量
就可以修改data里的数据值

总结

今天是第一天学习vue，虽然只是一个简单的helloworld,
但是已经体验到了Vue的甜头,可以不用去关注dom上的操作,
减少document.getxxx的操作,而是去关注数据上的操作
加油！
